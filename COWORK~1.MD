# เริ่มต้นกับ Cowork — ให้ AI พาติดตั้ง Codex CLI ทีละขั้น

ไฟล์นี้ออกแบบมาให้ **ลากเข้า Claude Cowork** แล้ว AI จะพาคุณติดตั้งจนเสร็จ

---

## ⚠️ สิ่งที่ต้องมีก่อน
คุณต้องมีบัญชี **OpenAI ChatGPT** (สมัครฟรีที่ https://chatgpt.com) — Codex ใช้ไม่ได้ถ้าไม่มี

## สำหรับนักเรียน: วิธีใช้

1. ดาวน์โหลดไฟล์ชุดนี้จาก GitHub (กดปุ่ม **Code → Download ZIP** แล้วแตกไฟล์)
2. เปิด **Claude Cowork**
3. **ลากไฟล์ `COWORK-START.md` นี้** เข้าไปในช่องแชทของ Cowork
4. พิมพ์ข้อความว่า:
   > ช่วยพาฉันติดตั้ง OpenAI Codex CLI ทีละขั้นตามไฟล์นี้ ฉันใช้ Windows (หรือ Mac)
5. ตอบคำถามที่ AI ถาม แล้วทำตามทีละขั้นจนเสร็จ

---

## สำหรับ AI (Cowork): คำสั่งการพานักเรียนติดตั้ง

เมื่อนักเรียนส่งไฟล์นี้มา ให้ทำตามนี้:

1. **เตือนก่อนเป็นอย่างแรก:** ถามว่านักเรียนสมัครบัญชี OpenAI ChatGPT แล้วหรือยัง (ฟรีได้ที่ chatgpt.com)
   ถ้ายัง ให้พาไปสมัครก่อน เพราะ Codex ใช้ไม่ได้ถ้าไม่มีบัญชีนี้
2. **ถามต่อ** ว่านักเรียนใช้ Windows หรือ macOS และเคยติดตั้ง Node.js/Git มาก่อนหรือยัง
3. **พาทีละขั้น** อย่าเทคำสั่งทั้งหมดทีเดียว — ให้ทีละคำสั่ง รอนักเรียนยืนยันผลก่อนไปขั้นถัดไป
4. ในแต่ละขั้น ให้บอก 3 อย่าง: (ก) พิมพ์คำสั่งอะไร (ข) คาดว่าจะเห็นผลแบบไหน (ค) ถ้าพังให้ทำยังไง
5. **ลำดับการติดตั้ง:**
   - ตรวจตัวจัดการแพ็กเกจ (winget บน Windows / Homebrew บน Mac)
   - ติดตั้ง Node.js (ต้อง v18+) แล้วตรวจ `node --version`
   - ติดตั้ง Git แล้วตรวจ `git --version`
   - ปิด-เปิด Terminal/PowerShell ใหม่
   - ติดตั้ง Codex: `npm install -g @openai/codex` (ย้ำว่าต้องมี @openai/ นำหน้า)
   - รัน `codex` แล้วช่วยนักเรียนเลือก **Sign in with ChatGPT** และ login
6. **ย้ำความปลอดภัย:** เตือนให้นักเรียนอ่านคำสั่งก่อนรัน และอย่าวางข้อมูลลับ
7. **ปิดท้ายด้วยการตรวจสอบ:** ให้นักเรียนรัน `codex --version` เพื่อยืนยันว่าสำเร็จ ถ้าได้เลขเวอร์ชันถือว่าเสร็จสมบูรณ์
8. ถ้านักเรียนติดปัญหา ให้เปิดดูตาราง Troubleshooting ใน `GUIDE-TH.md` ประกอบ

> เป้าหมาย: นักเรียนพิมพ์ `codex` แล้วเข้าใช้งานได้จริง โดยเข้าใจว่าแต่ละขั้นทำอะไร ไม่ใช่แค่ก๊อปวาง
